# A uniform driver for CAN device 

### Device supported

  - NI
    - USB-8473
  
  - SocketCan

  - ZLG(周立功) 
    - USBCAN-I/II
    - USBCANFD-100U
    - USBCANFD-mini
    - USBCANFD-200U
    - USNCANFD-400U
    - USBCANFD-800U

### Welcome to contribute for adding more device supported.
